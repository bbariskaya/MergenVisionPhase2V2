16. TRACKLET TEMPLATE AGGREGATION
==================================================

For each finalized raw tracklet:

1. Collect only recognition_eligible observations.
2. Require finite unit-normalized embeddings.
3. Divide the tracklet’s PTS span into up to five temporal bins.
4. Select the highest-quality observation per non-empty bin.
5. Fill remaining slots from highest-quality observations while respecting minimum PTS separation.
6. Choose a medoid using median pairwise cosine.
7. Calculate similarity-to-medoid values.
8. Reject embedding outliers using a deterministic median/MAD rule.
9. Compute a quality-weighted centroid.
10. L2 normalize the centroid.
11. Record every selected and rejected observation ID.
12. Record:
    - candidate count;
    - selected count;
    - rejected count;
    - min/median/max quality;
    - intra-tracklet cosine quantiles;
    - template norm;
    - failure reason if no valid template.

Default:

templates:
  max_selected_samples: 5
  min_selected_samples: 1
  min_temporal_separation_ns: 200000000
  outlier_mad_scale: 3.0
  outlier_absolute_cosine_floor: 0.20

A tracklet without a valid template still exists temporally but remains identity-unresolved.

==================================================
17. CANNOT-LINK AND OFFLINE RECONCILIATION
==================================================

Create cannot-link edges whenever two raw tracklets:

- contain observations in the same frame; or
- have overlapping appearance intervals beyond configured tolerance.

Cannot-link is absolute. Cosine similarity may not override it.

For candidate tracklet linking:

1. Compare valid tracklet templates.
2. Produce deterministic pair records.
3. Reject cannot-link pairs first.
4. Sort candidate edges by descending cosine, then stable tracklet IDs.
5. Merge components only if:
   - no cannot-link pair exists across components;
   - every cross-component template pair passes the minimum cosine threshold;
   - the merged normalized centroid remains above threshold for every member;
   - configured top1/top2 margin evidence passes;
   - deterministic tie-breaking is used.
6. Do not use unrestricted single-link clustering that can chain:
   A≈B, B≈C, A≠C.
7. Canonical track IDs must be deterministic for the frozen input/config.

Exploratory defaults:

reconciliation:
