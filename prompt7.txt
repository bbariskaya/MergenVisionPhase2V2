7. GERÇEK GSTREAMER/NVDEC AKIŞI
======================================================================

Friends.mp4 H.264 MP4 için gerçek pipeline:

  filesrc
    → qtdemux
    → h264parse
    → nvv4l2decoder
    → video/x-raw(memory:NVMM)
    → native consumer/appsink veya approved DeepStream pad path

Şartlar:

1. Decode element gerçekten `nvv4l2decoder` olmalı.
2. Output buffer NVMM olmalı.
3. `GstBuffer` içinden `NvBufSurface` alınmalı.
4. Surface GPU/device-mappable olmalı.
5. CPU system-memory video buffer görülürse structured failure:
   `VIDEO_GPU_SURFACE_CONTRACT_FAILED`
6. CPU fallback yapılmamalı.
7. Her frame için:
   - frame index
   - decode order
   - presentation timestamp
   - duration
   - source dimensions
   - rotation/display metadata
   korunmalı.
8. Zaman hesabı yalnız `frame_index / fps` üzerinden yapılmamalı.
9. PTS monotonicity ve VFR davranışı test edilmeli.
10. GStreamer ERROR/EOS event’leri düzgün ele alınmalı.
11. Pipeline cancel sırasında bounded sürede NULL state’e alınmalı.
12. GstBuffer/NvBufSurface/CUDA resource ömürleri RAII ile yönetilmeli.

Batch:

- RetinaFace için en fazla 8 frame batch.
- Batch dolmadan EOS gelirse partial batch flush.
- Her output `source_batch_index` üzerinden frame/PTS’ye geri bağlanmalı.
- Recognizer yüzleri detection sırasını koruyarak en fazla 32’lik chunk’larda çalıştırmalı.
- Hiçbir detection sessizce düşürülmemeli.
- Configured safety limit aşılırsa structured failure/rejection count üret.

8. FACE PIPELINE
======================================================================

Her frame batch’i için:

1. NV12/RGBA device surface’ı detector tensor’a CUDA kernel ile dönüştür.
2. Aspect ratio korunarak letterbox uygula.
3. Scale/pad bilgilerini her source frame için tut.
4. RetinaFace TensorRT çalıştır.
5. Score/bbox/landmark decode işlemini GPU’da yap.
6. NMS GPU’da çalışsın.
7. Bbox ve landmark’ları original source coordinates’e doğru reverse-map et.
8. Bbox clamp et; invalid/NaN/zero-area sonuçları reject et.
9. 5-point landmark order model profile ile birebir doğrulanmalı.
10. Canonical ArcFace similarity transform ile 112×112 aligned crop üret.
11. GlintR100 preprocessing profile contract’a birebir uymalı.
12. GlintR100 dynamic face batch çalıştır.
13. 512-D embedding’i GPU’da L2 normalize et.
14. Host’a yalnız compact finalized metadata ve gerektiğinde normalized embedding kopyala.

Zorunlu invariants:

- bbox source frame sınırları içinde.
- 5 landmark finite.
- detector score finite ve [0,1].
- embedding tam 512 eleman.
- embedding finite.
- eligible embedding normu tolerance içinde 1.0.
- batch 1 ve batch N sonuçları kabul edilebilir numeric tolerance içinde aynı.
- frame batch/chunk size değişikliği output order ve tracking sonucunu değiştirmemeli.

RetinaFace coordinate mapping/alignment hatasını model hatası diye geçiştirme.

----------------------------------------------------------------------
G. APPSINK BOUNDED BACKPRESSURE
----------------------------------------------------------------------

`nvstreammux → appsink` statik olarak link edilir.

Appsink:

  emit-signals = FALSE
  sync = FALSE
  max-buffers = 2
  drop = FALSE

olmalı.

Caps:

  video/x-raw(memory:NVMM), format=NV12

olarak sınırlandırılmalı.

`drop=FALSE` zorunludur. Inference yavaşlarsa decoder/pipeline bounded biçimde backpressure almalıdır; frame sessizce düşmemelidir.

C++ consumer loop:

1. `gst_app_sink_try_pull_sample()` bounded timeout ile çağrılır.
2. Timeout’ta cancellation flag ve GStreamer bus kontrol edilir.
3. Sample geldiyse `GstBuffer` alınır.
4. `NvBufSurface` ve `NvDsBatchMeta` çıkarılır.
5. FacePipeline batch çalıştırılır.
6. Tracker frame sırasıyla update edilir.
7. Observation writer’a yazılır.
8. CUDA batch işi bitmeden sample/buffer release edilmez.

Python’a frame callback yapılmaz.

----------------------------------------------------------------------
H. NVBUFSURFACE VE NVDSBATCHMETA OKUMA
----------------------------------------------------------------------

Her appsink sample için:

  GstBuffer* buffer = gst_sample_get_buffer(sample);

Önce:

  NvDsBatchMeta* batch_meta =
      gst_buffer_get_nvds_batch_meta(buffer);

kontrol edilir.

`batch_meta == nullptr` ise:

  VIDEO_BATCH_META_MISSING

hatası üret.

`GstBuffer` map yalnız `NvBufSurface` descriptor’ına erişmek içindir; pixel plane’i CPU’ya kopyalamak için değildir.

Logical pattern:

  GstMapInfo map_info{};
  gst_buffer_map(buffer, &map_info, GST_MAP_READ);
  NvBufSurface* surface =
      reinterpret_cast<NvBufSurface*>(map_info.data);

`NvDsBatchMeta.frame_meta_list` üzerinde iterate et.

Her `NvDsFrameMeta` için:

  batch_id = frame_meta->batch_id
  surface_params = surface->surfaceList[batch_id]

kullan.

`batch_id` bounds kontrolü:

  0 <= batch_id < surface->batchSize

Her `DeviceImageView` şunları içermeli:

- `surface_params.dataPtr`
- width
- height
- pitch
- color format
- layout/memory type
- frame_meta->batch_id
- frame_meta->source_id
- frame_meta->pad_index
- frame_meta->frame_num
- frame_meta->buf_pts
- source_frame_width
- source_frame_height

Önemli:

- Batched `GstBuffer` PTS’sini bütün framelere kopyalama.
- Her frame için `NvDsFrameMeta.buf_pts` kullan.
- `NvDsFrameMeta` içindeki original source dimensions bilgisini koru.
- `NvBufSurfaceSyncForCpu()` çağırma.
- Pixel plane’i host pointer olarak dereference etme.
- `cudaMemcpyDeviceToHost` ile full frame kopyalama.
- CPU RGB/NV12 buffer oluşturma.

Memory invariant:

  decoder cudadec-memtype == device
  streammux nvbuf-memory-type == cuda-device
  surface memType device-compatible
  dataPtr non-null
  colorFormat supported NV12
  pitch valid

Bunlardan biri bozulursa CPU fallback yerine fail et.

----------------------------------------------------------------------
I. CUDA ASENKRON ÖMÜR KURALI
----------------------------------------------------------------------

`NvBufSurface` decoder/mux buffer pool’una aittir.

CUDA kernel ve TensorRT inference devam ederken `GstSample`/`GstBuffer` release edilirse upstream aynı surface’ı yeniden kullanabilir ve corruption oluşur.

Bu nedenle:

1. İlgili `GstSample` batch boyunca retain edilir.
2. FacePipeline aynı configured CUDA stream’de preprocess/inference/alignment/embed çalıştırır.
3. Batch sonunda CUDA event record edilir.
4. Buffer release edilmeden event completion garanti edilir.
5. Frame başına `cudaDeviceSynchronize()` yapılmaz.
6. Basit ilk implementation’da batch başına stream/event synchronize kabul edilir.
7. Sonraki optimizasyonda completion callback/event queue ile buffer async release edilebilir.

Tek frame başına sync yasak; batch başına güvenli ownership barrier zorunludur.

EOS durumunda:

4. protobuf/zstd kapatılır.

6. pipeline NULL yapılır.

Cancel durumunda:

1. atomic cancellation flag set edilir.
2. pull loop en geç bounded timeout sonunda uyanır.
3. EOS beklenmeden pipeline NULL yapılabilir.
4. tracker/artifact `cancelled` footer veya discarded partial olarak kapanır.
5. canonical MinIO promote yapılmaz.
6. process exit code structured cancellation olur.

Teardown exact sıra:

1. pipeline state → NULL
2. pending samples/buffers release
3. mux request pads release
4. bus unref
5. elements/pipeline unref
6. CUDA events/streams/contexts release
7. TensorRT contexts release
8. local partial files cleanup
